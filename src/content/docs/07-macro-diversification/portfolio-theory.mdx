## 기대수익률과 위험

개별 자산의 **기대수익률** 은 각 시나리오별 수익률과 확률의 가중평균입니다:

$$E(R_i) = \sum_{s} p_s \cdot R_{i,s}$$

**예시**: 경기 호황(확률 30%, 수익률 20%), 보통(확률 50%, 수익률 10%), 불황(확률 20%, 수익률 -5%)인 자산의 기대수익률:

$$E(R_i) = 0.30 \times 20\% + 0.50 \times 10\% + 0.20 \times (-5\%) = 6\% + 5\% - 1\% = 10\%$$

**위험(Risk)** 은 표준편차 $\sigma_i$로 측정합니다:

$$\sigma_i^2 = \sum_{s} p_s \cdot [R_{i,s} - E(R_i)]^2$$

위 예시의 분산:

$$\sigma_i^2 = 0.30(20-10)^2 + 0.50(10-10)^2 + 0.20(-5-10)^2 = 30 + 0 + 45 = 75$$

표준편차: $\sigma_i = \sqrt{75} \approx 8.66\%$

두 자산 간의 **공분산**:

$$\text{Cov}(R_i, R_j) = \sum_{s} p_s \cdot [R_{i,s} - E(R_i)][R_{j,s} - E(R_j)]$$

**상관계수**:

$$\rho_{ij} = \frac{\text{Cov}(R_i, R_j)}{\sigma_i \sigma_j}$$

상관계수는 $-1 \leq \rho_{ij} \leq +1$ 범위를 가집니다.

## 포트폴리오의 기대수익률과 분산

두 자산 포트폴리오(비중 $w_1$, $w_2 = 1 - w_1$):

$$E(R_p) = w_1 E(R_1) + w_2 E(R_2)$$

$$\sigma_p^2 = w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\rho_{12}\sigma_1\sigma_2$$

**계산 예시**: 자산 A ($E(R_A) = 12\%$, $\sigma_A = 15\%$), 자산 B ($E(R_B) = 8\%$, $\sigma_B = 10\%$), $\rho_{AB} = 0.2$, $w_A = 0.6$, $w_B = 0.4$:

기대수익률: $E(R_p) = 0.6 \times 12\% + 0.4 \times 8\% = 7.2\% + 3.2\% = 10.4\%$

분산: $\sigma_p^2 = (0.6)^2(15)^2 + (0.4)^2(10)^2 + 2(0.6)(0.4)(0.2)(15)(10)$
$= 81 + 16 + 14.4 = 111.4$

표준편차: $\sigma_p = \sqrt{111.4} \approx 10.55\%$

단순 가중평균 위험 $= 0.6 \times 15\% + 0.4 \times 10\% = 13\%$보다 낮음 → 분산효과 확인

## 상관계수와 분산효과

상관계수에 따른 분산효과:

| 상관계수 | 분산효과 | 포트폴리오 위험 |
|----------|---------|---------------|
| $\rho = +1$ | 없음 | 가중평균과 동일 |
| $-1 < \rho < +1$ | 부분적 분산 | 가중평균보다 작음 |
| $\rho = 0$ | 독립적 분산 | 상당한 위험 감소 |
| $\rho = -1$ | 완전 분산 | 위험 완전 제거 가능 |

상관계수가 낮을수록 분산효과가 커지며, $\rho = -1$일 때 특정 비중에서 포트폴리오 위험을 0으로 만들 수 있습니다.

$\rho = -1$인 경우 위험 0을 만드는 비중:

$$w_1^* = \frac{\sigma_2}{\sigma_1 + \sigma_2}, \quad w_2^* = \frac{\sigma_1}{\sigma_1 + \sigma_2}$$

<Aside type="note" title="분산 투자의 한계">
자산 수를 늘려도 제거 가능한 비체계적 위험은 이론적으로 약 20~30개 종목 이후 거의 사라집니다. 그러나 모든 자산에 공통적으로 영향을 미치는 **체계적 위험(시장 위험)** 은 분산으로 제거할 수 없습니다.
</Aside>

## 마코위츠 평균-분산 모형

**해리 마코위츠(Harry Markowitz, 1952)** 는 투자자가 기대수익률과 분산만으로 포트폴리오를 평가한다고 가정하였습니다.

**주요 가정**:
- 투자자는 기대수익률을 극대화하고 분산을 최소화하는 **평균-분산 기준** 을 따름
- 단일 기간 투자 모형
- 자산 수익률은 정규분포를 따름
- 모든 투자자는 동일한 정보에 접근 가능

투자자는 동일한 위험 수준에서 높은 수익률을, 동일한 수익률에서 낮은 위험을 선호합니다.

**지배 원리(Dominance Principle)**: 포트폴리오 A가 B보다 같은 위험에 높은 수익률이거나 같은 수익률에 낮은 위험이면, A가 B를 지배합니다. 지배당한 포트폴리오는 합리적 투자자가 선택하지 않습니다.

## 효율적 프론티어와 최소분산포트폴리오

**효율적 프론티어(Efficient Frontier)** 는 주어진 위험 수준에서 최대 기대수익률을 제공하는 포트폴리오들의 집합입니다. 효율적 프론티어 아래에 위치한 포트폴리오는 **지배당한(Dominated)** 포트폴리오입니다.

**최소분산포트폴리오(MVP, Minimum Variance Portfolio)** 는 두 자산 포트폴리오에서 분산을 최소화하는 비중입니다:

$$w_1^* = \frac{\sigma_2^2 - \text{Cov}(R_1, R_2)}{\sigma_1^2 + \sigma_2^2 - 2\text{Cov}(R_1, R_2)}$$

**계산 예시**: $\sigma_1 = 15\%$, $\sigma_2 = 10\%$, $\rho_{12} = 0.2$이면 $\text{Cov} = 0.2 \times 15 \times 10 = 30$

$$w_1^* = \frac{100 - 30}{225 + 100 - 60} = \frac{70}{265} \approx 0.264$$

즉, 자산 1에 26.4%, 자산 2에 73.6% 배분할 때 분산이 최소가 됩니다.

<Aside type="tip" title="효율적 프론티어의 실무 활용">
실무에서는 수백 개 자산의 분산-공분산 행렬을 추정하여 수치 최적화로 효율적 프론티어를 계산합니다. 단, 공분산 추정 오류가 최적 비중에 큰 영향을 미치므로, **블랙-리터만(Black-Litterman) 모형** 같은 강건한 최적화 기법이 활용됩니다.
</Aside>

## 체계적·비체계적 위험과 분산 가능성

포트폴리오의 총위험은 두 부분으로 구성됩니다:

$$\sigma_p^2 = \text{체계적 위험} + \text{비체계적 위험}$$

- **체계적 위험(Systematic Risk)**: 금리, 경기, 인플레이션 등 시장 전체에 영향을 미치는 위험. 분산으로 제거 불가
- **비체계적 위험(Unsystematic Risk, Idiosyncratic Risk)**: 개별 기업이나 산업 고유의 위험. 분산으로 제거 가능

$n$개 동일 비중 포트폴리오의 분산:

$$\sigma_p^2 = \frac{1}{n}\bar{\sigma}_i^2 + \frac{n-1}{n}\overline{\text{Cov}}$$

$n \to \infty$이면 $\sigma_p^2 \to \overline{\text{Cov}}$ (평균 공분산, 즉 체계적 위험만 남음)

## 무위험자산 결합과 자본배분선(CAL)

무위험자산(수익률 $R_f$, 분산 0)과 위험 포트폴리오 $P$를 결합하면 **자본배분선(CAL, Capital Allocation Line)** 이 형성됩니다:

$$E(R_C) = R_f + \frac{E(R_P) - R_f}{\sigma_P} \cdot \sigma_C$$

CAL의 기울기는 **샤프 비율(Sharpe Ratio)** 입니다:

$$\text{Sharpe Ratio} = \frac{E(R_P) - R_f}{\sigma_P}$$

샤프 비율이 높을수록 단위 위험당 초과수익이 크므로 우수한 포트폴리오입니다.

**계산 예시**: $E(R_P) = 12\%$, $\sigma_P = 18\%$, $R_f = 3\%$일 때

$$\text{Sharpe Ratio} = \frac{12\% - 3\%}{18\%} = 0.5$$

투자자가 $\sigma_C = 9\%$ (위험의 절반)를 원한다면:
- 무위험자산 50%, 위험 포트폴리오 50% 투자
- 기대수익률: $3\% + 0.5 \times 9\% = 7.5\%$

## 접선포트폴리오와 분리정리

효율적 프론티어와 CAL이 접하는 지점이 **접선포트폴리오(Tangency Portfolio)** 입니다. 이 포트폴리오는 샤프 비율을 최대화하는 위험자산 포트폴리오입니다.

**분리정리(Separation Theorem)**: 모든 투자자가 동일한 기대를 가진다고 가정하면, 최적 위험자산 포트폴리오(접선포트폴리오)는 동일합니다. 투자자는 자신의 위험선호도에 따라 무위험자산과 접선포트폴리오의 비중만 조정하면 됩니다.

| 위험선호도 | 무위험자산 비중 | 접선포트폴리오 비중 |
|-----------|--------------|-----------------|
| 위험 회피 (보수적) | 높음 (예: 70%) | 낮음 (예: 30%) |
| 중립 | 중간 (예: 30%) | 중간 (예: 70%) |
| 위험 추구 (공격적) | 0% 이하 (레버리지) | 100% 이상 |

분리정리가 성립하면, 접선포트폴리오는 **시장포트폴리오(Market Portfolio)** 와 일치하며 이것이 CAPM의 기초가 됩니다.
