## 자본자산가격결정모형(CAPM)

**CAPM(Capital Asset Pricing Model)** 은 자산의 기대수익률이 체계적 위험(베타)에 의해 결정된다는 균형 모형입니다:

$$E(R_i) = R_f + \beta_i[E(R_m) - R_f]$$

여기서 $R_f$는 무위험이자율, $E(R_m)$은 시장포트폴리오 기대수익률, $E(R_m) - R_f$는 **시장위험프리미엄** 입니다.

**CAPM의 주요 가정**:
- 모든 투자자는 평균-분산 기준으로 최적화
- 투자자들은 동질적 기대(Homogeneous Expectations)를 가짐
- 세금, 거래비용 없음
- 무위험이자율로 자유롭게 차입·대출 가능
- 모든 자산은 거래 가능하며 완전히 분산 가능

## 베타와 체계적 위험

**베타($\beta$)** 는 시장 수익률 변화에 대한 자산 수익률의 민감도입니다:

$$\beta_i = \frac{\text{Cov}(R_i, R_m)}{\sigma_m^2} = \rho_{im} \cdot \frac{\sigma_i}{\sigma_m}$$

| 베타 | 해석 |
|------|------|
| $\beta = 1$ | 시장과 동일한 움직임 |
| $\beta > 1$ | 시장보다 크게 변동(공격적 자산) |
| $\beta < 1$ | 시장보다 작게 변동(방어적 자산) |
| $\beta = 0$ | 시장과 무관(무위험자산) |
| $\beta < 0$ | 시장과 반대 방향(헤지 역할) |

## 체계적 위험과 비체계적 위험

총위험은 두 부분으로 분해됩니다:

$$\sigma_i^2 = \beta_i^2 \sigma_m^2 + \sigma_{\epsilon_i}^2$$

- **체계적 위험(Systematic Risk)**: $\beta_i^2 \sigma_m^2$. 분산 불가, 시장 전체에 영향을 미치는 위험
- **비체계적 위험(Unsystematic Risk)**: $\sigma_{\epsilon_i}^2$. 분산 가능, 개별 자산 고유의 위험

분산투자를 통해 비체계적 위험은 제거 가능하므로, 시장은 체계적 위험에 대해서만 초과수익을 보상합니다.

## 증권시장선(SML)

**증권시장선(SML, Security Market Line)** 은 베타와 기대수익률의 선형 관계를 나타냅니다.

- SML **위**: 저평가된 자산 (기대수익률 > CAPM 요구수익률)
- SML **아래**: 고평가된 자산 (기대수익률 < CAPM 요구수익률)
- **알파($\alpha$)**: 실제 기대수익률과 SML 상의 수익률의 차이. $\alpha > 0$이면 초과 성과

**SML vs CAL 비교**:

| 구분 | CML(자본시장선) | SML(증권시장선) |
|------|---------------|---------------|
| X축 | 총위험($\sigma$) | 체계적 위험($\beta$) |
| 적용 대상 | 효율적 포트폴리오 | 모든 자산·포트폴리오 |

## 차익거래가격결정이론(APT)

**APT(Arbitrage Pricing Theory)** 는 스티븐 로스(Stephen Ross)가 제안한 다팩터 모형으로, 자산 수익률이 여러 공통 팩터에 의해 결정된다고 가정합니다:

$$R_i = E(R_i) + b_{i1}F_1 + b_{i2}F_2 + \cdots + b_{ik}F_k + \epsilon_i$$

균형 조건에서 기대수익률:

$$E(R_i) = R_f + b_{i1}\lambda_1 + b_{i2}\lambda_2 + \cdots + b_{ik}\lambda_k$$

여기서 $\lambda_k$는 팩터 $k$에 대한 위험프리미엄, $b_{ik}$는 팩터 민감도(팩터 베타)입니다.

## CAPM vs APT 비교

| 구분 | CAPM | APT |
|------|------|-----|
| 팩터 수 | 1개(시장포트폴리오) | 복수의 팩터 |
| 핵심 가정 | 평균-분산 최적화, 동질적 기대 | 차익거래 기회 없음 |
| 시장포트폴리오 | 명시적으로 필요 | 불필요 |
| 적용 유연성 | 낮음 | 높음 |
| 실증적 검증 | 어려움 | 팩터 선택 주관적 |

APT는 CAPM보다 가정이 덜 엄격하지만, 팩터의 수와 종류를 이론이 특정하지 않는 단점이 있습니다.
