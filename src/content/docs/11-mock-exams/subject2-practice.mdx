## 2과목 핵심 요약

### 주식 가치평가 핵심 공식

$$\text{DDM(고든 모형)} = \frac{D_1}{k - g} = \frac{D_0(1+g)}{k - g}$$

$$\text{CAPM} = R_f + \beta(R_m - R_f)$$

$$\text{PEG} = \frac{PER}{\text{이익성장률}(\%)}$$

### 채권 분석 핵심 공식

$$\text{수정듀레이션} = \frac{\text{맥컬리 듀레이션}}{1 + y}$$

$$\frac{\Delta P}{P} \approx -D^* \cdot \Delta y + \frac{1}{2} \cdot C \cdot (\Delta y)^2$$

### 파생상품 핵심 공식

$$\text{선물 이론가격} = S_0 \times (1 + r - d)^T$$

$$\text{풋-콜 패리티: } C + PV(X) = P + S_0$$

### 성과측정 핵심 공식

$$\text{샤프비율} = \frac{R_p - R_f}{\sigma_p}$$

$$\text{트레이너비율} = \frac{R_p - R_f}{\beta_p}$$

$$\text{젠센의 알파} = R_p - [R_f + \beta_p(R_m - R_f)]$$

$$\text{정규분포 VaR} = \mu - z \cdot \sigma$$

## 2과목 집중 문제

### 문제 1

고든 성장 모형(DDM)을 이용하여 주식의 내재가치를 구하려 한다. 현재 배당금(D₀)이 1,000원, 배당성장률(g)이 5%, 요구수익률(k)이 10%일 때 이 주식의 내재가치는?

1. 10,000원
2. 20,000원
3. 21,000원
4. 22,000원

<Aside type="tip" title="정답: 3번">
D₁ = D₀ × (1+g) = 1,000 × 1.05 = 1,050원
내재가치 = D₁ / (k - g) = 1,050 / (0.10 - 0.05) = 1,050 / 0.05 = **21,000원**
</Aside>

### 문제 2

CAPM에서 무위험수익률이 3%, 시장기대수익률이 10%, 베타(β)가 1.5인 주식의 기대수익률은?

1. 10.5%
2. 13.5%
3. 15.0%
4. 13.0%

<Aside type="tip" title="정답: 2번">
CAPM: E(R) = R_f + β × (R_m - R_f)
= 3% + 1.5 × (10% - 3%)
= 3% + 1.5 × 7%
= 3% + 10.5% = **13.5%**
</Aside>

### 문제 3

채권의 맥컬리 듀레이션이 4년이고 만기수익률(YTM)이 5%일 때 수정듀레이션은?

1. 3.64년
2. 3.81년
3. 4.00년
4. 4.20년

<Aside type="tip" title="정답: 2번">
수정듀레이션 = 맥컬리 듀레이션 / (1 + y)
= 4 / (1 + 0.05)
= 4 / 1.05 ≈ **3.81년**
</Aside>

### 문제 4

금리가 0.5%p 상승할 때 수정듀레이션 3년인 채권 가격의 변화율은 약 얼마인가?

1. +1.5%
2. -1.5%
3. +3.0%
4. -3.0%

<Aside type="tip" title="정답: 2번">
ΔP/P ≈ -D* × Δy = -3 × 0.005 = -0.015 = **-1.5%**
금리 상승 시 채권 가격은 하락하며, 듀레이션이 길수록 가격 변동이 크다.
</Aside>

### 문제 5

풋-콜 패리티(Put-Call Parity) 식으로 옳은 것은?

1. C - P = S₀ - PV(X)
2. C + PV(X) = P + S₀
3. P = C + S₀ - PV(X)
4. C = P - S₀ + PV(X)

<Aside type="tip" title="정답: 2번">
풋-콜 패리티: **C + PV(X) = P + S₀**
이를 변형하면 C = P + S₀ - PV(X), 또는 P = C - S₀ + PV(X)로 쓸 수 있습니다.
</Aside>

### 문제 6

샤프비율에 대한 설명으로 옳은 것은?

1. 체계적 위험(베타)만을 기준으로 성과를 측정한다.
2. 총위험(표준편차) 대비 초과수익률로 측정한다.
3. 벤치마크 대비 초과수익률로 측정한다.
4. 시장 포트폴리오와 비교할 때만 의미가 있다.

<Aside type="tip" title="정답: 2번">
샤프비율 = (포트폴리오 수익률 - 무위험수익률) / 포트폴리오 표준편차
총위험(표준편차) 대비 초과수익률을 측정합니다. 트레이너비율은 베타 대비, 정보비율은 추적오차 대비 초과수익률을 사용합니다.
</Aside>

### 문제 7

포트폴리오 수익률 12%, 무위험수익률 3%, 포트폴리오 베타 1.2, 시장 수익률 9%일 때 젠센의 알파는?

1. +0.6%
2. +1.8%
3. +2.4%
4. +3.0%

<Aside type="tip" title="정답: 2번">
CAPM 기대수익률 = 3% + 1.2 × (9% - 3%) = 3% + 7.2% = 10.2%
젠센의 알파 = 실제수익률 - CAPM 기대수익률
= 12% - 10.2% = **1.8%**
</Aside>

### 문제 8

VaR(Value at Risk)에 대한 설명으로 옳지 않은 것은?

1. 특정 신뢰수준에서 일정 기간 동안 발생할 수 있는 최대 손실액이다.
2. 신뢰수준이 높을수록 VaR 값은 커진다.
3. VaR는 꼬리 위험(tail risk)을 완전히 반영한다.
4. 정규분포 가정 하에서 95% 신뢰수준은 z=1.645를 사용한다.

<Aside type="tip" title="정답: 3번">
VaR는 정규분포를 가정할 경우 꼬리 위험(극단적 손실)을 과소평가할 수 있습니다. 이를 보완하기 위해 CVaR(조건부 VaR, Expected Shortfall)를 사용합니다.
</Aside>

### 문제 9

볼록성(Convexity)의 역할에 대한 설명으로 옳은 것은?

1. 금리 변화에 따른 채권 가격 변화를 선형으로 근사한다.
2. 듀레이션만으로 설명되지 않는 가격 변화의 오차를 보정한다.
3. 볼록성이 클수록 금리 상승 시 가격 하락이 더 크다.
4. 볼록성은 채권 만기와 무관하다.

<Aside type="tip" title="정답: 2번">
듀레이션은 가격-금리 관계를 선형으로 근사하지만, 실제 관계는 볼록(convex)합니다. 볼록성은 이 오차를 보정하며, 볼록성이 클수록 금리 하락 시 가격 상승이 더 크고 금리 상승 시 가격 하락이 더 작습니다.
</Aside>

### 문제 10

헤지펀드의 특징으로 옳지 않은 것은?

1. 공매도, 레버리지, 파생상품 등 다양한 전략을 사용한다.
2. 일반 공모펀드에 비해 규제가 엄격하다.
3. 절대수익(absolute return)을 추구하는 경향이 있다.
4. 운용 보수 외에 성과 보수(performance fee)를 부과하는 경우가 많다.

<Aside type="tip" title="정답: 2번">
헤지펀드는 사모 형태로 운용되어 일반 공모펀드보다 규제가 완화되어 있습니다. 이로 인해 다양한 투자 전략을 자유롭게 구사할 수 있습니다.
</Aside>

### 문제 11

채권의 만기수익률(YTM)에 대한 설명으로 옳은 것은?

1. 채권을 현재 시장가격으로 구입하여 만기까지 보유할 때 얻는 연평균 수익률이다.
2. 표면이자율과 항상 동일하다.
3. 채권 가격이 상승하면 YTM도 상승한다.
4. 만기수익률은 이표 재투자 수익률과 무관하다.

<Aside type="tip" title="정답: 1번">
YTM은 채권의 모든 현금 흐름(이표+원금)을 현재 시장가격과 일치시키는 할인율입니다. 채권 가격이 상승하면 YTM은 하락하며, 이표 재투자 수익률이 YTM과 같다는 가정을 내포합니다.
</Aside>

### 문제 12

주가수익비율(PER)에 대한 설명으로 옳은 것은?

1. PER가 낮을수록 항상 저평가된 주식이다.
2. PER = 주가 / 주당순이익(EPS)으로 계산한다.
3. PER은 성장주에 대해 적용하기 어렵다.
4. PER는 업종에 관계없이 동일한 기준으로 비교한다.

<Aside type="tip" title="정답: 2번">
PER = 주가 / EPS입니다. PER가 낮다고 반드시 저평가는 아니며(저성장·고위험일 수 있음), 업종별 비교가 적절합니다. PER는 성장주 분석에도 활용(PEG비율 등)됩니다.
</Aside>

### 문제 13

선물(Futures) 계약에 대한 설명으로 옳지 않은 것은?

1. 거래소에서 표준화된 조건으로 거래된다.
2. 매일 정산(일일정산, Mark-to-Market)이 이루어진다.
3. 계약 이행을 보증하기 위해 증거금을 납입한다.
4. 계약 당사자 간 직접 협상으로 조건이 결정된다.

<Aside type="tip" title="정답: 4번">
선물은 거래소에서 표준화된 계약으로 거래됩니다. 계약 조건을 당사자 간 협상으로 결정하는 것은 선물이 아닌 선도(Forward) 계약의 특징입니다.
</Aside>

### 문제 14

리츠(REITs)에 대한 설명으로 옳은 것은?

1. 부동산에 직접 투자하므로 유동성이 매우 낮다.
2. 배당 가능 이익의 일정 비율 이상을 의무적으로 배당해야 한다.
3. 개인 투자자는 리츠에 투자할 수 없다.
4. 모든 리츠는 비상장 형태로만 운영된다.

<Aside type="tip" title="정답: 2번">
리츠(부동산투자신탁)는 부동산에 투자하고 배당 가능 이익의 90% 이상을 배당해야 법인세 혜택을 받습니다. 거래소에 상장된 리츠는 개인 투자자도 주식처럼 매매할 수 있습니다.
</Aside>

### 문제 15

기술적 분석에 대한 설명으로 옳은 것은?

1. 기업의 재무제표와 가치평가를 중심으로 분석한다.
2. 과거 가격과 거래량 데이터를 이용하여 미래 가격을 예측한다.
3. 효율적 시장가설이 성립하면 기술적 분석의 효과가 더 높아진다.

<Aside type="tip" title="정답: 2번">

### 문제 16

분산투자의 효과에 대한 설명으로 옳은 것은?

1. 체계적 위험(시장 위험)을 완전히 제거할 수 있다.
2. 비체계적 위험(개별 위험)은 종목 수 증가로 감소시킬 수 있다.
3. 모든 자산의 상관계수가 1이어도 분산효과가 발생한다.

<Aside type="tip" title="정답: 2번">
분산투자로 제거 가능한 위험은 비체계적(개별) 위험이며, 체계적 위험(시장 전체 위험)은 분산으로 제거되지 않습니다. 상관계수가 1이면 분산효과가 없습니다.
</Aside>

### 문제 17

전환사채(CB)에 대한 설명으로 옳은 것은?

1. 발행 기업의 주식으로 전환할 수 있는 권리가 내재된 채권이다.
2. 일반 채권보다 표면이자율이 높다.
3. 전환권 행사 후에도 채권으로서의 효력이 유지된다.

### 문제 18

정보비율(Information Ratio)에 대한 설명으로 옳은 것은?

1. 포트폴리오의 총위험 대비 절대수익률을 측정한다.
2. 벤치마크 대비 초과수익률을 추적오차(tracking error)로 나눈 값이다.
3. 무위험수익률 대비 초과수익률을 측정한다.
4. 정보비율이 음수이면 항상 해당 펀드를 청산해야 한다.

<Aside type="tip" title="정답: 2번">
정보비율 = 초과수익률(α) / 추적오차(TE)
벤치마크 대비 얼마나 일관성 있게 초과수익을 창출하는지 측정합니다. 샤프비율은 무위험수익률 대비, 정보비율은 벤치마크 대비 성과를 측정합니다.
</Aside>

### 문제 19

옵션의 델타(Delta)에 대한 설명으로 옳은 것은?

1. 기초자산 가격이 1원 변할 때 옵션 가격의 변화량이다.
2. 콜옵션의 델타는 항상 음수이다.
3. 풋옵션의 델타는 항상 양수이다.
4. 델타는 시간이 지나도 변하지 않는다.

<Aside type="tip" title="정답: 1번">
델타는 기초자산 가격 1단위 변화에 대한 옵션 가격의 변화량입니다. 콜옵션 델타는 0~1(양수), 풋옵션 델타는 -1~0(음수) 범위입니다. 델타는 시간, 변동성 등에 따라 변합니다.
</Aside>

### 문제 20

면역전략(Immunization)에 대한 설명으로 옳은 것은?

1. 채권 포트폴리오의 듀레이션을 투자 목표 기간과 일치시켜 금리 변동 위험을 헤지한다.
2. 금리 변동이 없을 때만 효과적인 전략이다.
3. 채권의 볼록성을 최소화하여 금리 위험을 제거한다.
4. 면역전략은 한 번 설정하면 재조정이 필요 없다.

<Aside type="tip" title="정답: 1번">
면역전략은 포트폴리오의 듀레이션을 투자 목표 기간과 일치시켜 금리 변동에 따른 재투자 위험과 가격 위험이 서로 상쇄되도록 합니다. 금리 변동 후에도 효과를 유지하려면 주기적 재조정이 필요합니다.
</Aside>
