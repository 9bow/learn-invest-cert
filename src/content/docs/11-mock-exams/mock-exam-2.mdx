### 문제 1

맥컬리 듀레이션(Macaulay Duration)이 4년인 채권의 **수정 듀레이션(Modified Duration)**은? (YTM = 5%, 연 1회 이자 지급)

1. 3.60년
2. 3.81년
3. 4.00년
4. 4.20년

<Aside type="tip" title="정답: 2번">
수정 듀레이션 = 맥컬리 듀레이션 / (1 + YTM)
= 4 / (1 + 0.05)
= 4 / 1.05
= **3.81년**
수정 듀레이션은 금리 1% 변화 시 채권 가격의 근사 변화율(%)을 나타낸다.
</Aside>

### 문제 2

주식 A의 베타가 0.8, 무위험수익률 2%, 시장 기대수익률 10%일 때 CAPM에 의한 **균형 기대수익률**은?

1. 6.4%
2. 8.0%
3. 8.4%
4. 10.0%

<Aside type="tip" title="정답: 3번">
E(Ri) = Rf + βi × (E(Rm) − Rf)
= 2% + 0.8 × (10% − 2%)
= 2% + 0.8 × 8%
= 2% + 6.4% = **8.4%**
</Aside>

### 문제 3

다음 중 **풋-콜 패리티(Put-Call Parity)**를 올바르게 표현한 것은? (동일 기초자산, 동일 행사가격, 동일 만기 가정, 무배당)

1. C − P = S − PV(K)
2. C + P = S + PV(K)
3. C − P = PV(K) − S
4. C + P = PV(K) − S

<Aside type="tip" title="정답: 1번">
풋-콜 패리티: **C − P = S − PV(K)**
또는 동치로 C + PV(K) = P + S
여기서 C는 콜옵션 가격, P는 풋옵션 가격, S는 기초자산 현재가격, PV(K)는 행사가격의 현재가치이다.
</Aside>

### 문제 4

다음 중 **효율적 시장가설(EMH)**의 **강형(Strong Form)**이 성립할 때 의미하는 바로 옳은 것은?

1. 과거 가격 정보를 이용한 초과수익은 불가능하다.
2. 공시된 공개 정보를 이용한 초과수익은 불가능하다.
3. 내부자 정보를 포함한 모든 정보가 가격에 반영되어 어떤 정보로도 초과수익이 불가능하다.
4. 기술적 분석을 이용하면 초과수익이 가능하다.

<Aside type="tip" title="정답: 3번">
EMH의 세 가지 형태: 약형(과거 가격 정보), 준강형(모든 공개 정보), **강형(내부자 정보 포함 모든 정보)**. 강형이 성립하면 내부자도 초과수익을 낼 수 없다. 현실에서는 강형이 성립하지 않는 경우가 많아 내부자거래 규제가 필요하다.
</Aside>

### 문제 6

다음 중 **전환사채(Convertible Bond, CB)**에 대한 설명으로 옳지 않은 것은?

1. 일정 조건 하에 주식으로 전환할 수 있는 권리가 부여된 사채이다.
2. 전환권이 내재되어 있어 일반 회사채보다 낮은 쿠폰금리로 발행된다.

### 문제 7

펀드의 **순자산가치(NAV, Net Asset Value)**를 구하는 공식으로 옳은 것은?

1. (총자산 + 총부채) / 발행 수익증권 좌수
2. (총자산 − 총부채) / 발행 수익증권 좌수
3. 총자산 / 발행 수익증권 좌수
4. 총부채 / 발행 수익증권 좌수

<Aside type="tip" title="정답: 2번">
NAV(단위: 1좌당) = **(총자산 − 총부채) / 발행 수익증권 좌수**
집합투자기구의 순자산은 보유 자산의 시가에서 차입금 등 부채를 차감한 금액이며, 이를 총 좌수로 나누어 1좌당 기준가격을 산출한다.
</Aside>

### 문제 8

다음 중 **ELS(주가연계증권)**의 특징으로 옳은 것은?

1. 원금이 법적으로 100% 보장된다.
2. 기초자산 가격 변동에 연계하여 수익이 결정되는 파생결합증권이다.

### 문제 9

**자본시장법**상 투자매매업과 투자중개업의 차이점에 대한 설명으로 옳은 것은?

1. 투자매매업자는 고객 명의로, 투자중개업자는 자기 명의로 매매한다.
2. 투자매매업자는 자기 명의·자기 계산으로, 투자중개업자는 타인의 계산으로 매매를 중개한다.
3. 두 업무는 자본시장법상 동일하게 취급된다.
4. 투자중개업은 자기 계산으로 금융투자상품을 매매하는 업무이다.

<Aside type="tip" title="정답: 2번">

### 문제 10

주식의 **기술적 분석**에서 이동평균선의 **골든크로스(Golden Cross)**가 의미하는 것은?

1. 단기 이동평균선이 장기 이동평균선을 위에서 아래로 돌파하는 시점
2. 단기 이동평균선이 장기 이동평균선을 아래에서 위로 돌파하는 시점
3. 주가가 이동평균선 위로 올라가는 시점
4. 거래량이 이동평균선을 상향 돌파하는 시점

<Aside type="tip" title="정답: 2번">
골든크로스는 **단기 이동평균선이 장기 이동평균선을 아래에서 위로 돌파**하는 현상으로, 일반적으로 강세 전환 신호로 해석된다. 반대로 단기선이 장기선을 위에서 아래로 돌파하는 것은 데드크로스(Dead Cross)로 약세 신호로 본다.

### 문제 11

**CDS(신용부도스왑, Credit Default Swap)**에 대한 설명으로 옳은 것은?

1. 보장 매수자는 프리미엄을 수취하고 신용사건 발생 시 손실을 보상해 준다.
2. CDS 스프레드가 상승하면 준거자산의 신용도가 개선된 것으로 해석된다.
3. 보장 매도자는 프리미엄을 지급하고 신용사건 발생 시 손실을 보상받는다.
4. CDS는 채권 보유자의 신용위험 헤지 수단으로 활용된다.

<Aside type="tip" title="정답: 4번">
CDS에서 **보장 매수자(Protection Buyer)**는 프리미엄을 지급하고 준거기업의 신용사건(디폴트 등) 발생 시 손실을 보상받아 신용위험을 헤지한다. CDS 스프레드 상승은 신용도 **악화**를 의미한다. 따라서 보기 4번이 옳고 1·2·3번은 틀린 설명이다.
</Aside>

### 문제 12

다음 중 **블랙-숄즈 모형(Black-Scholes Model)**의 가정으로 옳지 않은 것은?

1. 기초자산 가격은 기하 브라운 운동을 따른다.
2. 옵션 만기까지 배당이 없다.
3. 거래비용과 세금이 없다.
4. 변동성은 만기와 행사가격에 따라 다르다.

<Aside type="tip" title="정답: 4번">
블랙-숄즈 모형은 **변동성이 일정(constant volatility)**하다고 가정한다. 현실에서는 변동성이 만기와 행사가격에 따라 달라지는 변동성 스마일/스큐 현상이 나타나 모형의 한계로 지적된다.
</Aside>

### 문제 13

다음 포트폴리오 성과 측정 지표 중 **체계적 위험(베타)**만을 위험 척도로 사용하는 것은?

1. 샤프비율(Sharpe Ratio)
2. 트레이너비율(Treynor Ratio)
3. 정보비율(Information Ratio)
4. 소르티노비율(Sortino Ratio)

<Aside type="tip" title="정답: 2번">
트레이너비율 = (Rp − Rf) / **βp**로, 분모에 포트폴리오의 베타(체계적 위험)를 사용한다. 샤프비율은 전체 표준편차, 정보비율은 추적오차, 소르티노비율은 하방 표준편차를 분모로 사용한다.
</Aside>

### 문제 14

다음 중 **양적완화(QE, Quantitative Easing)**에 대한 설명으로 옳은 것은?

1. 중앙은행이 기준금리를 인상하여 시중 유동성을 흡수하는 정책이다.

<Aside type="tip" title="정답: 3번">
양적완화는 기준금리가 이미 제로(0) 수준에 도달했을 때 중앙은행이 국채, MBS 등 장기 자산을 직접 매입하여 시중에 유동성을 공급하는 **비전통적 통화완화 정책**이다. 2008년 글로벌 금융위기 이후 미 연준 등이 적극 활용하였다.

### 문제 15

### 문제 17

다음 중 **리츠(REITs, Real Estate Investment Trusts)**에 대한 설명으로 옳은 것은?

1. 부동산에 간접 투자하는 상품으로, 투자자에게 임대수익 등을 배당한다.
2. 모든 리츠는 비상장 형태로 운용된다.
3. 투자 대상이 부동산에 한정되어 있어 주식·채권에는 투자할 수 없다.
4. 부동산 직접 투자와 달리 소액 투자가 불가능하다.

<Aside type="tip" title="정답: 1번">
리츠는 다수 투자자로부터 자금을 모아 부동산 또는 부동산 관련 증권에 투자하고 임대수익, 매각차익 등을 배당 형태로 분배하는 간접투자 수단이다. 상장 리츠(K-REIT 등)는 거래소에서 거래되며 소액 투자도 가능하다.
</Aside>

### 문제 18

다음 중 **채권 신용등급**에서 투자등급과 투기등급(고수익)의 경계선에 해당하는 등급은? (S&P 기준)

1. AAA
2. BBB−
3. BB+
4. CCC

<Aside type="tip" title="정답: 2번 또는 3번">
S&P 기준으로 **BBB−** 이상이 투자등급(Investment Grade)이고, **BB+** 이하가 투기등급(Speculative Grade, High Yield)이다. 경계선은 BBB−(최하 투자등급)와 BB+(최상 투기등급) 사이이다. 시험에서는 BBB−를 투자등급의 최하한으로 묻는 경우가 많다.
</Aside>

### 문제 19

다음 중 **자본시장법상 공매도(Short Selling)**에 관한 설명으로 옳은 것은?

1. 차입 없이 주식을 매도하는 무차입 공매도는 국내에서 허용된다.
2. 차입 공매도는 주식을 빌려서 매도한 후 나중에 되갚는 방식이다.
3. 공매도는 주가 하락 시 손실이 발생한다.
4. 공매도는 시장 유동성을 낮추는 효과가 있다.

<Aside type="tip" title="정답: 2번">
차입 공매도는 주식을 차입하여 매도한 후, 나중에 주식을 매수하여 상환하는 방식이다. 국내에서는 **무차입 공매도(Naked Short Selling)**는 원칙적으로 금지된다. 공매도는 주가 **상승** 시 손실이 발생하며, 유동성 제공과 가격 발견 기능에 기여한다는 시각도 있다.
</Aside>

### 문제 20

다음 중 **금리 위험(Interest Rate Risk)**을 관리하는 수단으로 가장 적절하지 않은 것은?

1. 이자율 선물 매도
2. 이자율 스왑 (변동금리 수취, 고정금리 지급)
3. 주가지수 선물 매수
4. 금리 캡(Interest Rate Cap) 매수

<Aside type="tip" title="정답: 3번">
주가지수 선물은 **주식시장 위험(시장위험)**을 관리하는 수단으로, 금리 위험 헤지에는 직접적으로 활용되지 않는다. 금리 위험 관리 수단으로는 이자율 선물·옵션, 이자율 스왑, 금리 캡·플로어 등의 파생상품이 있다.
</Aside>
