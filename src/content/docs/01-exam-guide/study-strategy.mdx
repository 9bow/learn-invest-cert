## 추천 학습 순서

시험 과목을 공부하는 순서는 합격률에 큰 영향을 미칩니다. 권장 순서는 다음과 같습니다.

**2과목 → 3과목 → 1과목**

이 순서를 권장하는 이유는 다음과 같습니다.

- **2과목을 먼저**: 투자분석과 리스크관리는 난이도가 높고 계산 문제가 많습니다. 머리가 가장 맑을 때 집중적으로 학습하면 이해도가 높아집니다.
- **3과목을 두 번째로**: 문항 수가 50개로 가장 많습니다. 합격 여부를 좌우하는 핵심 과목이므로 충분한 시간을 투자해야 합니다. 법규는 암기, 투자운용 전략은 계산과 이해가 병행됩니다.
- **1과목을 마지막으로**: 금융상품과 세제는 비교적 암기 위주입니다. 2과목과 3과목에서 금융상품 기초 개념을 먼저 익힌 뒤 학습하면 중복 학습 없이 효율적으로 정리할 수 있습니다.

## 기간별 학습 플랜

### 4주 단기 플랜 (이미 금융 관련 배경 지식 보유자)

| 주차 | 학습 내용 | 목표 |
|------|----------|------|
| 1주차 | 2과목 전체 (투자분석, 리스크관리, 대안투자, 해외투자) | 핵심 개념과 공식 숙지 |
| 2주차 | 3과목 전반부 (직무윤리, 법규) | 법규 체계와 주요 조항 암기 |
| 3주차 | 3과목 후반부 (주식·채권·파생·거시경제) | 계산 문제 반복 연습 |
| 4주차 | 1과목 + 기출문제 전 과목 반복 | 약점 보완, 실전 감각 유지 |

### 8주 표준 플랜 (일반 수험생 권장)

| 주차 | 학습 내용 |
|------|----------|
| 1~2주차 | 2과목: 투자분석기법, 리스크관리 |
| 3주차 | 2과목: 대안투자, 해외증권투자 + 2과목 기출 풀기 |
| 4~5주차 | 3과목: 직무윤리, 법규 |
| 6~7주차 | 3과목: 투자운용 전략 I (주식·채권·파생·거시경제) |
| 8주차 | 1과목 + 전 과목 기출 2회독 |

### 12주 여유 플랜 (처음 시작하는 수험생)

| 기간 | 학습 내용 |
|------|----------|
| 1~3주차 | 2과목: 대안투자, 해외증권투자, 투자분석기법 |
| 4~5주차 | 2과목: 리스크관리 + 2과목 문제풀이 |
| 6~8주차 | 3과목: 직무윤리, 법규 |
| 9~11주차 | 3과목: 투자운용 전략 I + 거시경제 |
| 12주차 | 1과목 + 전 과목 모의고사 |

## 이론 대 문제풀이 비율

권장 비율은 **이론 3 : 문제풀이 7** 입니다.

투자자산운용사 시험은 이론 이해를 바탕으로 한 응용 문제가 많습니다. 이론만 공부하고 문제를 소홀히 하면 실제 시험에서 낯선 표현과 선택지에 당황할 수 있습니다. 반드시 기출문제를 충분히 풀어 문제 유형에 익숙해져야 합니다.

- 첫 번째 이론 학습 후 즉시 해당 단원 문제풀이
- 전 과목 학습 완료 후 기출문제 최소 2회독
- 틀린 문제는 반드시 오답노트 작성

## 과목별 학습 팁

**2과목 (투자분석 및 리스크관리)**
- VaR 계산 공식(정규분포, 역사적 시뮬레이션, 몬테카를로)을 반드시 암기
- 기술적 분석 지표(RSI, MACD, 볼린저 밴드)는 공식보다 해석 방법에 집중
- 대안투자는 PEF 구조(GP/LP)와 헤지펀드 전략 분류를 표로 정리

**3과목 (법규·투자운용·거시경제)**
- 자본시장법 조문은 조번호보다 내용 이해를 우선
- 채권 듀레이션과 볼록성(Convexity) 계산 문제는 유형이 반복되므로 패턴 익히기
- 옵션 가격결정(풋-콜 패리티) 공식은 반드시 유도 과정 이해

**1과목 (금융상품 및 세제)**
- 세제 개정 사항을 매년 확인(금융소득종합과세 기준, 비과세 한도 등)
- ELS·DLS 구조와 조기상환 조건을 그림으로 그려 이해

## 시험 당일 전략

- 시험 시작 전 공식 정리 노트를 한 번 더 확인
- 문제 순서대로 풀되, **모르는 문제는 표시 후 다음 문제로** 이동
- 시간 배분: 1문항당 평균 72초, 계산 문제에 최대 3분까지만 허용
- **과목별 40% 과락 기준**을 항상 의식하면서 취약 과목 문제를 먼저 집중 공략
- 시험 종료 10분 전 마킹 상태 최종 확인
